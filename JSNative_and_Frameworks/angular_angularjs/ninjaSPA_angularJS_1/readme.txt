одностраничное приложение создано на основе туториола по angularjs youtobe-канала the net ninja

Взаимодействие между компонентами осуществяется с помощью роутера.

На главной странице появляется случайный ниньзя.

В компоненте list ninjas можно добавлять\удалять ниньзь из списка. Также присутствует примеры сортировки.

Также есть анимация добавления\удаления элементов из списка. 

В компоненте contact form находиться пример формы с валидацией используя angJS.
